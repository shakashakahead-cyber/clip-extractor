クリップ抽出くん ベータ (Clip Extractor Beta)
================================================================================

このツールは、動画内の「盛り上がり（笑い声や歓声）」をAIが自動検知し、
ハイライトシーンを抽出・切り抜きするソフトウェアです。

【重要：使用前の準備】
本ソフトを使用するには「FFmpeg」という外部ツールが必要です。
以下の手順でインストールを行ってください。

1. FFmpegのダウンロード
   以下のサイトから Windows用のビルドをダウンロードします。
   https://www.gyan.dev/ffmpeg/builds/
   (「ffmpeg-git-full.7z」などでOKです)

2. 解凍と配置
   ダウンロードしたファイルを解凍し、中の「bin」フォルダにある
   「ffmpeg.exe」を取り出します。

3. パスの設定 (推奨)
   ffmpeg.exe を C:\ffmpeg などのわかりやすい場所に置き、
   Windowsの「システム環境変数の編集」から「Path」にそのフォルダを追加してください。

   ※ または、ffmpeg.exe をこのソフト（動画ハイライト抽出くんβ.exe）と
      同じフォルダに直接置くだけでも動作します（こちらの方が簡単です）。

【使い方】
1. アプリを起動します。
2. 「動画を選択」ボタンで解析したい動画ファイルを選びます。
3. 自動的にAI解析が始まり、ハイライト候補がリストアップされます。
4. 必要なシーンにチェックを入れ、「書き出し」ボタンで保存します。

【GPU利用について】
高性能なAMD/NVIDIAグラフィックボードを搭載している場合、
自動的にGPUを使用して高速に解析を行います。
（GPUがない場合や、スペックが低い場合はCPUで動作しますが、解析に時間がかかる場合があります）

【ライセンス】
本ソフトはオープンソースコンポーネントを使用しています。
詳細は添付の THIRD_PARTY_NOTICES.txt をご確認ください。
